Excel Data Exploration Assignment - 2
Project Overview: This project focuses on data cleaning and preparation using Microsoft Excel. The dataset was processed to improve data quality, consistency, and readiness for analysis.
1. Handling Missing Values:Identified missing values in numerical and categorical columns. Applied appropriate imputation techniques and manual validation to ensure data completeness.
2.  Standardizing Text Data: Reviewed product names and category fields for inconsistent formatting and spelling errors.  Used Excel text-cleaning tools and Find & Replace to standardize values across the dataset.
3. Duplicate Record Analysis:  Identified and highlighted duplicate records for further validation.  Records were retained since repeated entries may represent valid business transactions rather than data duplication.
4. Splitting and Merging Data:  Performed data transformations by separating combined fields and merging relevant columns where required.
5. Data Formatting:  Applied consistent formatting to improve readability and maintain a standardized dataset structure.
6. Conditional Formatting:  Used conditional formatting rules to highlight important values, patterns, and potential data quality issues.
